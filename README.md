# 高股息 ETF 智慧決策面板 & 雲端自適應複利回測系統 (AI 最佳化完全體)

[Streamlit App](https://share.streamlit.io/) 
Python Version: 3.10+
Database: Supabase
Automation: GitHub Actions
Notification: LINE Notify

這是一個專為台灣高股息 ETF 設計的全方位量化投資決策面板。本專案擺脫傳統單機運作的限制，整合了網頁前端、雲端資料庫、定時排程與即時通訊軟體，打造出一個不需要電腦開機、24 小時在線運作的量化交易 SaaS 平台。

經由 Google Colab 實驗室大數據地毯式參數最佳化（Grid Search）調校，本系統已全面升級為**「個股自適應基因大腦」**，捨棄傳統一刀切的僵化指標，改為每檔 ETF 量身打造專屬的黃金操盤聖經。

線上即時戰情面板：https://my-etf-radar-x9zih5qspvgsgnywx6rick.streamlit.app/#dd19d17e

## 核心功能與技術亮點

1. **四大天王個股自適應基因大腦 (New! 🚀)**
   - 透過歷史數據多維度轟炸，破解四大高股息 ETF 的真實投資個性，動態載入各自最高勝率、最高報酬的專屬黃金參數，拒絕死板公式。

2. **100% 同步的真實複利回測引擎**
   - 完美還原 Colab 實驗室模型，採用真實的資產翻滾複利模型，非錯誤的單利加總。
   - 整合 `auto_adjust=True` 雙保險數據校正與最大歷史天數（Max Period）下載，徹底解決 Yahoo Finance 數據殘缺 Bug，真實還原歷史除權息權值。

3. **商用級 Supabase 雲端資料庫緩存**
   - 網頁開啟時優先讀取雲端資料庫，避免高頻率請求 Yahoo Finance API 導致被封鎖 IP（Rate Limit），看盤速度提升 300%。

4. **LINE Notify 雲端主動推播 (不需開機)**
   - 部署 GitHub Actions 雲端定時機器人，每天台股收盤後（下午 15:30）自動起床執行監控腳本。
   - 結合自適應訊號過濾，只在觸發實戰買賣點或強勢抱緊時發送高級結構化字卡（Flex Message），絕不發送觀望雜訊打擾。

本系統的核心操盤策略圍繞在**「除息後低檔撈底」**與**「多天高檔鈍化抱緊」**。經大數據回測，四大天王已自動校正為以下專屬參數組合：

* **🏢 00929 復華台灣科技優息 ➔ 【爆發型黑馬：33 買、3天快閃】**
  - **策略**：除息後 20 天內，5天內 KD 曾**跌破 33** 則於金叉買入；進入 82 以上強勢區後，**高檔鈍化滿 3 天**轉弱即獲利落袋。
  - **特點**：純科技股波段噴速快，大數據證實 3 天快閃最能鎖住暴利，滾出翻倍報酬！
* **🏢 00919 群益台灣精選高息 ➔ 【極致多頭王：39 買、5天死抱】**
  - **策略**：除息後 20 天內，5天內 KD 曾**跌破 39** 則於金叉買入；進入強勢區後，必須**高檔鈍化死死抱緊滿 5 天**才放手。
  - **特點**：股性極度強勢，大數據實測勝率高達 **100%**，長抱 5 天才能吃滿整條大牛市！
* **🏢 0056 元大高股息 ➔ 【沉穩老大哥：32 買、5天長抱】**
  - **策略**：除息後 20 天內，5天內 KD 曾**跌破 32** 則於金叉買入；進入強勢區後，同樣需**高檔鈍化滿 5 天**再出場。
  - **特點**：慢熱穩健，極其適合大數據調校出的 32 點除息極限修正低位，用 5 天長抱把利潤榨到最乾淨。
* **🏢 00878 國泰永續高股息 ➔ 【高頻率周轉王：39 買、3天小賺】**
  - **策略**：除息後 20 天內，5天內 KD 曾**跌破 39** 則於金叉買入；進入強勢區後，**高檔鈍化滿 3 天**即小賺落袋。
  - **特點**：抗震防禦力強、股性溫吞跌不深（買點放寬到 39 最剛好），最適合「一跌就買、一噴就跑」的高周轉密集操盤法。
