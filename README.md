# 高股息 ETF 智慧決策面板 & 雲端自適應複利回測系統 (動態 SaaS 完全體)

[Streamlit App](https://share.streamlit.io/) 
Python Version: 3.10+
Database: Supabase (PostgreSQL)
Automation: GitHub Actions
Notification: LINE Notify

這是一個專為台灣高股息 ETF 設計的全方位個人化量化投資決策 SaaS 平台。本專案擺脫傳統單機運作與寫死代碼的限制，整合了網頁前端、雲端自訂資料庫、定時排程與即時通訊軟體，打造出一個不需要電腦開機、24 小時在線運作的自動化投資生態系。

經由大數據地毯式參數最佳化（Grid Search）調校，本系統已全面升級為**「動態自適應基因大腦」**。不僅支援使用者「自訂精選追蹤清單」，更內建全市場熱門高股息 ETF 的專屬黃金操盤聖經！

線上即時戰情面板：https://my-etf-radar-x9zih5qspvgsgnywx6rick.streamlit.app/#dd19d17e

## 🚀 核心功能與技術亮點

1. **個人化精選股雲端管理後台 (New! 🔥)**
   - 前端整合 Supabase 雲端動態資料表（`user_watchlist`），使用者可於側邊欄即時「新增」或「刪除」追蹤標的。網頁頁籤、技術線圖與背景雷達將全自動動態渲染生成，實現真正的個人化看盤。

2. **全市場明星個股自適應基因大腦 (Expanded! 🧠)**
   - 破解多檔高股息 ETF 的天生投資個性，系統會根據當前選取的代號，自動載入 Colab 實驗室回測出的最高勝率、最高報酬專屬黃金參數。

3. **100% 同步的真實複利回測引擎**
   - 採用真實的資產翻滾複利模型，非錯誤的單利加總。
   - 整合 `auto_adjust=True` 雙保險數據校正與 5 分鐘極速快清機制（Cache TTL=300s），徹底防範 Yahoo Finance 盤後數據干擾與舊快取卡資料 Bug。

4. **LINE Notify 雲端定時主動推播 (不需開機)**
   - 部署 GitHub Actions 雲端定時工作流，每天台股傍晚資訊更新後（下午 17:30）自動起床執行監控腳本。
   - 只在觸發專屬實戰買賣點或強勢抱緊時發送高級結構化字卡（Flex Message），絕不發送觀望雜訊。
