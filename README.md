# 高股息 ETF 智慧決策面板 & 雲端複利回測系統 (完全體)

[Streamlit App](https://share.streamlit.io/) 
Python Version: 3.10+
Database: Supabase
Automation: GitHub Actions
Notification: LINE Notify

這是一個專為台灣高股息 ETF（如 00929、00919、0056、00878）設計的全方位量化投資決策面板。本專案擺脫傳統單機運作的限制，整合了網頁前端、雲端資料庫、定時排程與即時通訊軟體，打造出一個不需要電腦開機、24 小時在線運作的量化交易 SaaS 平台。

線上即時戰情面板：https://my-etf-radar-x9zih5qspvgsgnywx6rick.streamlit.app/#dd19d17e

---

## 核心功能與技術亮點

1. 100% 同步的真實複利回測引擎
   - 完美還原 Colab 實驗室模型，捨棄錯誤的單利加總，採用真實的資產翻滾複利模型。
   - 整合 auto_adjust=True 雙保險數據校正，自動排除盤中即時數據干擾，真實還原歷史除權息權值。

2. 獨創「除息撈底 ＆ 高檔鈍化抱緊」實戰策略
   - 波段撈底：嚴格鎖定除息日後 20 天內，當技術指標 KD 跌破 18 進入極度超賣區，並於黃金交叉時觸發買入訊號。
   - 飆速抱緊：當 KD 進入 82 以上強勢鈍化區時，啟動狂飆抱緊機制；直到連續 4 天高檔鈍化結束轉弱時，精準觸發賣出訊號。

3. 商用級 Supabase 雲端資料庫緩存
   - 網頁開啟時優先讀取雲端資料庫，避免高頻率請求 Yahoo Finance API 導致被封鎖 IP（Rate Limit）。
   - 實現數據跨平台同步，看盤速度提升 300%。

4. LINE Notify 雲端主動推播 (不需開機)
   - 部署 GitHub Actions 雲端定時機器人，每天台股收盤後（下午 15:30）自動起床執行監控腳本。
   - 一旦策略觸發實戰買賣點，立刻發送精美結構化訊息至使用者的 LINE 手機或群組，實現無感操盤。
